👉 THIS is likely your case (why it's not navigating)

Problem:

Dashboard is NOT changing tab index

✅ Fix: Use shared index controller
Step 1: Create global index (if not already)
ValueNotifier<int> navIndex = ValueNotifier(0);
Step 2: Use it in main screen
ValueListenableBuilder<int>(
  valueListenable: navIndex,
  builder: (context, index, _) {
    return IndexedStack(
      index: index,
      children: [
        HomePage(),
        TripsPage(),
        BillsPage(),
      ],
    );
  },
)
Step 3: FIX Dashboard click
InkWell(
  onTap: () {
    navIndex.value = 1; // Trips tab index
  },
  child: DashboardCard(title: "Trips")
)
🚨 IMPORTANT:
Do NOT use Navigator.push if app uses tabs
Always update tab index instead
